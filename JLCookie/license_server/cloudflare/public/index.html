ขั้นตอนเริ่มรันบอท

1 เตรียมโปรแกรมจำลอง Emulator

แนะนำให้ใช้ LDPlayer 9 หรือ MuMu Player 12 ตั้งค่าความละเอียดหน้าจอจำลองเป็น 1280x720 (DPI 240) ในระบบ Android ของโปรแกรมจำลอง

2 นำ License Key ไปใส่ในตัวบอท

เมื่อซื้อคีย์เรียบร้อยแล้ว คีย์จะปรากฏในเมนู "บัญชีของฉัน" ให้กดคัดลอกคีย์นั้นมาเปิดบอท (JL_Bot_Cookie.exe) แล้ววางในช่องสิทธิ์ใช้งาน จากนั้นกดเปิดใช้งาน

3 กดค้นหา Instance ของบอท

กดปุ่มรีเฟรช ⟳ ระบบจะทำหารหา Instance ของโปรแกรมจำลองในระบบให้อัตโนมัติ ให้ทำการเลือกเครื่องที่รันเกมอยู่

4 ตั้งค่าและรันบอท

เลือกตัวเลือกการรันและบูสต์ไอเทมในหน้าแท็บตัวเลือก จากนั้นกดปุ่ม "▶ เริ่มบอท" เพื่อสั่งให้บอทเริ่มการสแกนและควบคุมเกมอัตโนมัติทันที
