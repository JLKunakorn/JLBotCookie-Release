วิธีใช้งาน JLmain V1.0.3 Premium

1. สิ่งที่ต้องเตรียม
- เครื่อง Windows 10/11
- โปรแกรมอีมูเลเตอร์ LDPlayer หรือ MuMu
- เกมต้องติดตั้งและเข้าได้ตามปกติ
- License key ที่ได้รับจากร้าน

2. เปิดโปรแกรมครั้งแรก
- ดับเบิลคลิก JLmain_V1.0.3_Premium.exe
- ถ้ามีหน้าต่าง Windows Security หรือ Antivirus แจ้งเตือน ให้กดอนุญาตเฉพาะเมื่อไฟล์มาจากร้านโดยตรง
- โปรแกรมจะแสดงหน้าต่างใส่ License key
- ใส่ key ที่ได้รับ แล้วกดเปิดใช้งาน
- ถ้า key ถูกต้อง โปรแกรมจะแสดงเวลาคงเหลือของสิทธิ์ใช้งาน

3. เลือกอีมูเลเตอร์
- เปิดอีมูเลเตอร์และเปิดเกมให้พร้อมก่อน
- ไปที่แท็บ Setting
- เลือกชนิดอีมูเลเตอร์ เช่น LDPlayer หรือ MuMu
- กดปุ่มรีเฟรช
- เลือก Instance ที่ต้องการใช้งาน
- ถ้าหาเครื่องไม่เจอ ให้ปิด/เปิดอีมูเลเตอร์ใหม่ แล้วกดรีเฟรชอีกครั้ง

4. ตั้งค่าการเล่น
- กลับไปที่แท็บคอนโทรล
- ตั้งจำนวนรอบที่ต้องการให้บอททำงาน
- เลือกตัวเลือกการเล่นตามต้องการ
- ถ้าเลือกโหมด Multi ระบบจะจัดการสุ่มบูสต์ก่อนเริ่มตามเงื่อนไขของโหมดนั้น

5. เริ่มและหยุดบอท
- กด Start เพื่อเริ่มทำงาน
- ขณะทำงาน ปุ่มจะเปลี่ยนเป็น Stop
- ถ้าต้องการหยุด ให้กด Stop แล้วรอสักครู่ให้โปรแกรมหยุดคำสั่งที่กำลังทำอยู่

6. การดูสถานะ
- ช่อง Coin แสดงเหรียญล่าสุด/ยอดรวมที่อ่านได้
- ช่อง Log จะแสดงเฉพาะเหตุการณ์สำคัญ เช่น เชื่อมต่อเครื่อง, เริ่มรอบ, จบรอบ, อ่านเหรียญ, license หมดเวลา หรือข้อผิดพลาด
- ถ้าอ่านเหรียญไม่ได้ในบางรอบ โปรแกรมยังทำงานต่อได้ ให้เช็คภาพเกม/หน้าจอผลลัพธ์ว่าถูกบังหรือไม่

7. เมื่อ License หมดอายุ
- โปรแกรมจะหยุดให้ใช้งานตามเวลาที่กำหนด
- ติดต่อร้านเพื่อซื้อหรือต่ออายุ key ใหม่
- ถ้าเปลี่ยนเครื่อง อาจต้องให้ร้าน reset สิทธิ์เครื่องก่อนใช้งาน

8. วิธีแก้ปัญหาเบื้องต้น
- กด Start แล้วไม่ทำงาน: เช็คว่าเลือก Instance ถูกต้อง และอีมูเลเตอร์เปิดอยู่
- หาเครื่องไม่เจอ: กดรีเฟรชในหน้า Setting หรือเปิดอีมูเลเตอร์ใหม่
- License ใช้ไม่ได้: เช็คว่า key พิมพ์ถูกต้อง และเครื่องมีอินเทอร์เน็ต
- ขึ้นว่าใช้ครบจำนวนเครื่องแล้ว: ติดต่อร้านเพื่อ reset สิทธิ์เครื่อง
- โปรแกรมค้างหรือหน้าจอไม่อัปเดต: ปิดโปรแกรมแล้วเปิดใหม่

9. ข้อควรระวัง
- ห้ามเปิดโปรแกรมหลายตัวพร้อมกันกับ key เดียว ถ้าแพ็กเกจไม่ได้รองรับหลายเครื่อง
- ไม่ควรแก้ไขไฟล์โปรแกรมหรือใช้ไฟล์จากแหล่งอื่น
- ควรใช้ไฟล์ exe ที่ได้รับจากร้านโดยตรงเท่านั้น
