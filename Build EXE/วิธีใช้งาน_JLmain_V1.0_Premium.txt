วิธีใช้งาน JLmain V2.0.3 Premium

1. สิ่งที่ต้องเตรียม
- เครื่อง Windows 10/11
- โปรแกรมอีมูเลเตอร์ LDPlayer หรือ MuMu
- เกมต้องติดตั้งและเข้าได้ตามปกติ
- Premium License key ที่ได้รับจากร้าน

2. เปิดโปรแกรมครั้งแรก
- ดับเบิลคลิก JLmain_V2.0.3_Premium.exe
- ถ้ามีหน้าต่าง Windows Security หรือ Antivirus แจ้งเตือน ให้กดอนุญาตเฉพาะเมื่อไฟล์มาจากร้านโดยตรง
- ใส่ License key ที่ได้รับแล้วกดเปิดใช้งาน
- ถ้า Key ถูกต้อง โปรแกรมจะแสดงเวลาคงเหลือของสิทธิ์ใช้งาน

3. เลือกอีมูเลเตอร์และจอ
- เปิดอีมูเลเตอร์และเปิดเกมให้พร้อมก่อน
- ไปที่แท็บ Setting แล้วเลือก LDPlayer, MuMu หรือ LDPlayer + MuMu
- กดปุ่มรีเฟรชเพื่อโหลดเฉพาะ Instance ที่ออนไลน์จริง
- เลือกจอที่ต้องการใช้งาน จอหลักใช้ Key หลัก
- การเปิด AutoFarm เพิ่มแต่ละจอต้องเพิ่ม Premium Key อีก 1 Key ในหน้า License
- ถ้าหาเครื่องไม่เจอ ให้ปิด/เปิดอีมูเลเตอร์ใหม่ แล้วกดรีเฟรชอีกครั้ง

4. ตั้งค่าการเล่น
- ตั้งจำนวนรอบที่ต้องการให้บอททำงาน ค่า 0 คือวนต่อเนื่องจนกว่าจะสั่งหยุด
- เลือกโหมดการเล่น: กระโดดสุ่ม, สไลด์, กระโดด + สไลด์ หรือไม่ทำอะไร
- ตั้งช่วงดีเลย์กระโดด/สไลด์ได้ ค่าเริ่มต้นคือ 0.14-0.72 วินาที
- เปิด/ปิดรับหัวใจจาก Mail และรับ Relic พร้อมกำหนดจำนวนขั้นต่ำของ Mail ได้
- ถ้าเลือกโหมด Multi ระบบจะจัดการสุ่มบูสต์ก่อนเริ่มตามเงื่อนไขของโหมดนั้น

5. เริ่มและหยุดบอท
- เลือกจอแล้วกด Start เพื่อเริ่มทำงาน
- แต่ละจอเริ่ม/หยุดแยกกันได้
- ปุ่มหยุดทั้งหมดจะสั่งหยุด AutoFarm ทุกจอ

6. การดูสถานะ
- ช่อง Coin แสดงเหรียญล่าสุดและยอดรวมแยกตามจอ
- ยอดรวมสะสมตั้งแต่เริ่ม AutoFarm ครั้งนั้น และเริ่มจาก 0 เมื่อเริ่มงานครั้งใหม่
- ช่อง Log แสดงเฉพาะเหตุการณ์สำคัญ เช่น เชื่อมต่อเครื่อง เริ่มรอบ จบรอบ อ่านเหรียญ License หมดเวลา หรือข้อผิดพลาด
- ถ้าอ่านเหรียญไม่ได้ในบางรอบ โปรแกรมยังทำงานต่อได้ ให้เช็คว่าหน้าผลลัพธ์ถูกบังหรือไม่

7. การแจ้งเตือน Discord
- ไปที่ Setting > Discord Notification แล้วเลือก Webhook ของตัวเอง
- ใส่ Discord Webhook URL และเลือกแจ้งตอนเริ่ม จบรอบ และหยุด
- กดบันทึกแล้วกดทดสอบก่อนเริ่มใช้งานจริง
- Webhook ถูกเก็บแบบเข้ารหัสสำหรับผู้ใช้ Windows คนปัจจุบัน

8. เมื่อ License หมดอายุ
- โปรแกรมจะหยุดจอที่ใช้ Key หมดอายุ
- ติดต่อร้านเพื่อซื้อหรือต่ออายุ Key
- ถ้าเปลี่ยนเครื่อง อาจต้องให้ร้านรีเซ็ตสิทธิ์เครื่องก่อนใช้งาน

9. วิธีแก้ปัญหาเบื้องต้น
- กด Start แล้วไม่ทำงาน: เช็คว่าเลือกจอถูกต้อง เกมเปิดอยู่ และ Key ของจอนั้นยังใช้งานได้
- หาเครื่องไม่เจอ: กดรีเฟรชในหน้า Setting หรือเปิดอีมูเลเตอร์ใหม่
- License ใช้ไม่ได้: เช็คว่า Key พิมพ์ถูกต้องและเครื่องมีอินเทอร์เน็ต
- ขึ้นว่าใช้ครบจำนวนเครื่องแล้ว: ติดต่อร้านเพื่อรีเซ็ตสิทธิ์เครื่อง
- โปรแกรมค้างหรือหน้าจอไม่อัปเดต: หยุดงาน ปิดโปรแกรม แล้วเปิดใหม่

10. ข้อควรระวัง
- Premium Key 1 Key ใช้ AutoFarm ได้ 1 จอ การเพิ่มจอต้องใช้ Key แยก
- ไม่ควรแก้ไขไฟล์โปรแกรมหรือใช้ไฟล์จากแหล่งอื่น
- ควรใช้ไฟล์ EXE/ZIP ที่ได้รับจากร้านหรือหน้า Release ทางการเท่านั้น
